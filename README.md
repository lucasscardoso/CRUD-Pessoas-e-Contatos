# CRUD PESSOAS E CONTATOS

### para rodar a aplicação voce precisa ter instalado em sua maquina:

-**JDK 17.0.5**

-**Springboot 3.0.12**

-**Maven-3.9.1**

-**utilizado banco de dados MySql**
